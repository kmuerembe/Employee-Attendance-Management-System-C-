Descrição

Este projeto consiste em um Sistema de Presença de Funcionários desenvolvido em C++. O sistema permite o controlo de presença dos funcionários de uma empresa através de um ambiente simples e organizado.

Funcionalidades

- Login de Administrador.
- Login de Funcionário.
- Cadastro de funcionários.
- Registo de presença.
- Consulta de presenças.
- Consulta de faltas.
- Pesquisa por dia.
- Pesquisa por mês.
- Relatórios de presença.
- Controle de acesso por senha.

Tecnologias Utilizadas

- Linguagem C++
- Dev C++
- Programação estruturada
- Manipulação de arquivos

Estrutura do Sistema

Administrador

O administrador possui acesso completo ao sistema, podendo:

- Cadastrar funcionários.
- Consultar presenças.
- Gerar relatórios.
- Pesquisar por datas.
- Gerir os dados do sistema.

Funcionário

O funcionário possui acesso limitado, podendo:

- Fazer login.
- Registrar a sua presença.
- Consultar as suas informações.

Objetivo do Projeto

Desenvolver um sistema que facilite o controlo de assiduidade dos funcionários, aplicando conceitos de programação em C++.

Conceitos Aplicados

- Funções
- Estruturas de decisão (if, switch)
- Estruturas de repetição (for, while)
- Vetores
- Manipulação de arquivos
- Menus interativos

Como Executar

1. Abra o projeto no Dev C++.
2. Compile o código.
3. Execute o programa.
4. Escolha o tipo de utilizador.
5. Utilize as funcionalidades do sistema.
